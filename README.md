# Team-profile-generator
# Description 
A team profile generator using node.js object oriented programming and Javascript. 
# Installation 
Install node.js 
<br>
Run commands 
<br>
npm init
npm install 
npm i inquirer@8.2.4

# Usage 
run node index.js
